>Welcome,
>Here you will learn Some of the basics of HTML.
>HTML stands for Hyper Text Markup Language. HTML describes the structure of Web pages using markup.
>In HTML, you use tags to represent HTML elements. Those elements are the building blocks of HTML pages.
>Browsers do not display the HTML tags, however they do use them to render the content of the page.
>Below is a sample HTML document:
> <!DOCTYPE html>
> <html>
> <head>
> <title>Page Title</title>
> </head>
> <body>
> <h1>My First Heading</h1>
> <p>My first paragraph.</p>
> </body>
> </html>
>It would display the following:

My First Heading

My first paragraph.
