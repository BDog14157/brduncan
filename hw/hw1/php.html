>Welcome,
>Here you will learn Some of the basics of PHP.
>PHP is a recursive acronym for Hypertext Preprocessor. PHP is used as an open source general-purpose scripting
>language. It is very well suited for web developement. It can also be embedded into HTML.
>
>To use PHP on a page you must use start and end brackets, like a normal Div in HTML. Here is an example.
> <?php
>  echo "Hi, I'm a PHP script!";
> ?>
>The code above will display "Hi, I'm a PHP script!" on the page.
>
>As opposed to other programming languages such as HTML, which is client side, PHP is server side. This means that
>your browser is not interpreting it, rather the code is being run on a server and relayed
>back to your browser for you to view.
>Isn't that cool!
>
>One of the best reasons to use PHP is that the coding is made simple for those new to programming, but also has a
>plethora of features for professionals.
>
>One of the strongest and most useful features that PHP has is support for many types of databases. One of the more
>popular database extensions is mysql.
